# 模板  
~~~  
#version version_number  
in type in_variable_name;  
in type in_variable_name;  
  
out type out_variable_name;  
  
uniform type uniform_name;  
  
void main()  
{  
out_variable_name=weird_stuff_we_processed;  
}  
~~~  
glsl语法，类c  
允许通过uniform实现全局量，在各个着色器、主程序之间传递数据  
#  顶点着色器（vs）  
    
从VBO中获取数据，处理顶点位置、属性  
因而vs的输入需要特殊语句：  
layout (location = 0) in type in_variable_name;  
  
gl_Position不需要传入fs，即vs的设定结果  
#  片段着色器（fs）  
  
FragColor是fs的最终结果  
  
# 着色器类  
链接vs和fs成为shader program，封装成类以便创建多个类对象
