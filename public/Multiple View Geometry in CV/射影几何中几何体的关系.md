  
三维：  
无穷远平面$\pi_{\infty}$  
绝对二次曲线$\Omega_{\infty}$  
绝对二次曲面$Q_\infty$  
绝对对偶二次曲面$Q_\infty^*$（对偶绝对二次曲线，即绝对二次曲线的对偶）  
  
二维：  
无穷远线$l_{\infty}$  
虚圆点$I \& J$  
绝对二次曲线图像（IAC）$\omega$  
绝对二次曲线对偶图像（DIAC）$\omega^*$，也即对偶绝对二次曲线$C_{\infty}^*$ （或记为$\Omega_{\infty}^*$）  
  
映射：  
  
| 三维                | 外形线        | 图像         |  
| ----------------- | ---------- | ---------- |  
| $\Omega_{\infty}$ | $\omega^*$ | $\omega$   |  
| $Q_\infty^*$      | $\omega$   | $\omega^*$ |  
| $\pi_{\infty}$    |            | $l_\infty$ |  
注：这里可以把绝对二次曲线视为边二次曲面，并由此有外形线的概念  
  
  
  
三维中，  
$Q_\infty$交$\pi_{\infty}$于$\Omega_{\infty}$  
$\pi$交$\pi_{\infty}$于$l_{\infty}$  
$\pi$的$l_{\infty}$与$\Omega _{\infty}$交于两点，即$\pi$的虚圆点  
  
二次曲面的图像与外形线成对偶关系  
  
  
二维中，  
圆与$l_{\infty}$交于$I\&J$  
圆与$\omega$切于$I\&J$  
$\omega$与$l_{\infty}$交于$I\&J$  
  
绝对二次曲线只有虚点，映射为无穷远直线上的一些虚点，不具有实际映射结果
